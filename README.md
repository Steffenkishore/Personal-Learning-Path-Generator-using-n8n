# 🤖 Personal Learning AI Agent using n8n + Gemini + Google Workspace

An AI-powered Personal Learning Agent built using `n8n`, `Google Gemini`, `Google Docs`, `Google Calendar`, and `SerpAPI`.

This automation helps users generate a complete learning roadmap automatically. The AI agent researches resources, creates a structured learning path, stores it inside Google Docs, and schedules learning sessions directly into Google Calendar.

---

# 🚀 Features

- Generate personalized learning paths using AI
- Research learning resources automatically
- Create structured day-wise study plans
- Automatically create Google Docs
- Update learning plans dynamically
- Schedule learning sessions in Google Calendar
- Fully automated AI workflow using n8n

---

# 🛠️ Tech Stack

- Automation : n8n
- AI Model : Google Gemini
- Search Engine : SerpAPI
- Documentation : Google Docs API
- Scheduling : Google Calendar API

# ⚙️ How This Automation Works

## 1️⃣ User Gives a Learning Goal

The user enters a learning goal in the chat interface.

Example:

```text
I want to learn Full Stack Web Development in 30 days
```

The workflow starts automatically when a chat message is received.

---

## 2️⃣ AI Agent Receives the Request

The `AI Agent` node acts as the brain of the workflow.

It receives the user's learning goal and decides:

- What to learn
- How to structure the roadmap
- Which tools to use
- How to organize the learning schedule

---

## 3️⃣ Google Gemini Generates the Learning Plan

The `Google Gemini Chat Model` is connected to the AI Agent.

Gemini helps generate:

- Day-wise learning roadmap
- Learning topics
- Study order
- Task breakdown

Example:

```text
Day 1 → HTML Basics
Day 2 → CSS Fundamentals
Day 3 → Responsive Design
```

---

## 4️⃣ SerpAPI Researches Learning Resources

The AI Agent uses `SerpAPI` to search for:

- Tutorials
- Documentation
- Courses
- YouTube videos
- Articles

This helps create a more useful and resource-rich learning plan.

---

## 5️⃣ Google Docs Creates the Learning Document

The workflow automatically creates a Google Document containing:

- Learning roadmap
- Daily schedule
- Resources
- Study instructions

This allows users to access their learning plan anytime.

---

## 6️⃣ Google Docs Updates the Content

After creating the document, the workflow updates it with the complete structured learning content generated by the AI.

---

## 7️⃣ Google Calendar Schedules Learning Sessions

The workflow automatically creates calendar events for each learning session.

Example:

```text
Day 1 → HTML Basics → 6 PM
Day 2 → CSS Fundamentals → 6 PM
```

This helps users follow the learning plan consistently.

# 🧩 Steps to Create This Automation

## Step 1 — Create a Chat Trigger

Add the `When Chat Message Received` node in n8n.

This starts the workflow whenever the user sends a learning request.

---

## Step 2 — Add AI Agent Node

Add the `AI Agent` node and connect it to the chat trigger.

This node controls the workflow logic and tool usage.

---

## Step 3 — Connect Google Gemini

Connect the `Google Gemini Chat Model` to the AI Agent.

Model Used:

```text
gemini-2.0-flash
```

---

## Step 4 — Add SerpAPI

Connect `SerpAPI` to the AI Agent.

This allows the AI to search the internet for learning resources.

---

## Step 5 — Add Google Docs Nodes

Add:

- Create Document Node
- Update Document Node

These nodes create and fill the learning roadmap document automatically.

---

## Step 6 — Add Google Calendar Node

Add the `Create Event` node to schedule learning sessions automatically.

---

## Step 7 — Configure AI Agent Instructions

Inside the AI Agent node, provide system instructions explaining:

- How to create learning paths
- How to use tools
- When to create documents
- When to schedule calendar events

This helps the AI follow the correct workflow sequence.
